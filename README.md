Sou uma profissional da área de Engenharia e Qualidade de Software com uma trajetória sólida na garantia de estabilidade e performance de ecossistemas complexos: plataformas **Web**, **Mobile (iOS/Android)** e **Firmware**. Atualmente, expando meu escopo de atuação como **Desenvolvedora Full-Stack**, criando APIs REST robustas em **Java (Spring Boot)** e aplicações cliente dinâmicas em **React/Next.js com TypeScript**. Complementarmente, sou pós-graduanda em **Data Science e IA** na PUCRS, focando no uso inteligente de dados para otimizar processos de engenharia.
